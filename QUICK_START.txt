🚀 QUICK DEPLOYMENT (5 STEPS)
------------------------------

1. EXTRACT THE ZIP FILE
   - Unzip to your desired location
   - Navigate to the TysunMikePro folder

2. INSTALL DEPENDENCIES
   Open terminal in the project folder and run:
   $ npm install

3. SET UP ENVIRONMENT
   - Copy .env.example to .env
   - Edit .env with your credentials:
     * DATABASE_URL (PostgreSQL connection string)
     * JWT_SECRET (generate with: node -e "console.log(require('crypto').randomBytes(32).toString('hex'))")
     * SENDGRID_API_KEY (from SendGrid dashboard)
     * MAIL_FROM (your sender email)
     * BASE_URL (your domain or http://localhost:8080)

4. SET UP DATABASE
   - Create a PostgreSQL database
   - Run the schema:
     $ psql "your-database-url" < db.sql

5. START THE SERVER
   $ npm start
   
   Your website will be running at http://localhost:8080
